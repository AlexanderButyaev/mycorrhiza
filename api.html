The Dataset API reference¶

The “Myco” file reader class¶

class mycorrhiza.dataset.Myco(file_path, diploid: bool = True)¶

Mycorrhiza formatted file reader.

Parameters:
  • file_path (str) – Path to the file containing the data.
  • diploid (bool) – Dilpoid genotypes occupy 2 rows.

The “Structure” file reader class¶

class mycorrhiza.dataset.Structure(file_path: str, diploid: bool = True, n_optional_cols: int = 0, missing='-9')¶

Structure formatted file reader.

Parameters:
  • file_path (str) – Path to the file containing the data.
  • diploid (bool) – Dilpoid genotypes occupy 2 rows.
  • n_optional_cols (int) – Number of columns to ignore after the third column.
  • missing – Representation of missing data.

File conversions¶

mycorrhiza.dataset.vcf_to_structure(path_in: str, path_out: str, populations: list, diploid: bool = True)¶

Converts a VCF file into the STRUCTURE format.

Parameters:
  • path_in (str) – Path to the VCF file containing the data.
  • path_out (str) – Output file, where the result will be written.
  • populations (list) – List of population labels for the samples in the VCF file, in the same order.
  • diploid (bool) – If the samples are diploid.
